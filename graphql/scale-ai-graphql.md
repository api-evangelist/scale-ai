# Scale AI GraphQL API

Scale AI provides data labeling, RLHF, and AI evaluation services. The API covers task creation for labeling, annotation retrieval, workforce management, evaluation datasets, and model improvement pipelines.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Scale AI REST/gRPC API
**Documentation:** https://scale.com/docs/
